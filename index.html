Book Wishlist API

Descrição

API REST desenvolvida em ASP.NET Core para gerenciamento de uma lista de desejos de livros. A aplicação permite adicionar, atualizar, consultar e remover livros da lista, utilizando boas práticas de arquitetura para organização e manutenção do código.

Tecnologias utilizadas

  • C#
  • ASP.NET Core Web API
  • Entity Framework Core
  • SQLite
  • AutoMapper

Padrões e boas práticas

  • Repository Pattern para abstração do acesso a dados
  • DTO Pattern para separar modelos de domínio da API
  • AutoMapper para mapear entidades e DTOs
  • Estrutura organizada em camadas

Funcionalidades

  • Adicionar um livro à lista de desejos
  • Consultar todos os livros cadastrados
  • Consultar livro por ID
  • Atualizar informações de um livro
  • Remover livro da lista
